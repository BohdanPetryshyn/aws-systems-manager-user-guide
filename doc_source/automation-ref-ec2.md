# Amazon EC2<a name="automation-ref-ec2"></a>

AWS Systems Manager Automation provides predefined runbooks for Amazon Elastic Compute Cloud\. Runbooks for Amazon Elastic Block Store are located in the [Amazon EBS](automation-ref-ebs.md) section of the runbook reference\. For more information about runbooks, see [Working with runbooks](automation-documents.md)\. For information about how to view runbook content, see [View runbook content](automation-documents-reference.md#view-automation-json)\.
